Eigenvalue computations

 var (eVecs, eVals) = eig(X) produces a  matrix eVals of eigenvalues and a   full matrix eVecs whose columns are the corresponding eigenvectors so  that eVals*X = eVecs*X.
 

Since there are some small differences at the interface of each library we consider an example for each one, to highlight the differences.

    1.   Matrix class    (one indexed NUMAL based class)

The eVals matrix is NX2, first column has the real values and the second column the imagenary values of each eigenvector.

var myMat = M1("5 3 -1; 4 3 2; -4 -1.2 4") // create an one -indexed Matrix
var (eigsVec, eigsVal) = eig(myMat)

eigsVec: scalaSci.Matrix = 
 
-0.639  -0.748  -0.335  
1.000  -0.232  0.341  
-0.323  1.000  1.000  

eigsVal: scalaSci.Matrix = 
 
-0.201  0.000  
7.269  0.000  
4.932  0.000  

// verify eigenvalue equation
var ev1 =  eigsVec(':',1)   // first eigenvector
var xeVal1 = myMat*ev1
var xeVal2 = eigsVal(1,1)*ev1

var  diff = xeVal1 - xeVal2  // should be zero



2.   Mat class (scalaSci.Mat, zero indexed JAMA based class)
This class returns only the real values of the eigenvalues in a diagonal matrix. 

Example:
var myMat0 = M("5 3 -1; 4 3 2; -4 -1.2 4") // create a zero-indexed Mat
var (eigsVec0, eigsVal0) = eig(myMat0) 

eigsVec0: scalaSci.Mat = 
 0.589  -0.520  0.591  
0.183  0.814  -0.600  
-0.787  -0.263  -1.761  
eigsVal0: scalaSci.Mat = 
7.269  0.000  0.000  
0.000  -0.201  0.000  
0.000  0.000  4.932  

// verify eigenvalue equation
var ev0 =  eigsVec0(':',0)   // first eigenvector
var xeVal10 = myMat0*ev0
var xeVal20 = eigsVal0(0,0)*ev0

var  diff0 = xeVal10 - xeVal20  // should be zero
