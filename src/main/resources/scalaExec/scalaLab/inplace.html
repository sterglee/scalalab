In place operations


Operators ++, --, **, /|  implement in-place addition, subtraction, multiplication and division.
The in-place operators are implemented by avoiding creating a copy of the receiver's data area and a new object, thus they are memory and time efficient, generally more than  3 times faster from the corresponding +, -, *, / 

Examples


Example 1
var v = vones(10);   // create a Vector of 10 elements
v--30   // subtract 30 from all its elements
Result
v: scalaSci.Vec = 

-29.000  -29.000  -29.000  -29.000  -29.000  -29.000  -29.000  -29.000  -29.000  -29.000  
Note that the contents of v are altered


Example 2
var m = ones(2,3)  // create an one-indexed Matrix
m ++ 100  // add 100 to all its elements
Result
m: scalaSci.Matrix = 
 101.000  101.000  101.000  
101.000  101.000  101.000  
Note that the contents of m are altered
Example 3
Associativity of  ++, --, **, /= is the same as +, -, **, / respectively, for example:
var mm = ones(2,2) ++ 100*2
Result
mm: scalaSci.Matrix = 
 
201.000  201.000  
201.000  201.000  
i.e. first the multiplication * operator is evaluated and then the ++ operator

Example 4
var m = zeros0(3,4)  
var mm = 5+(m—1)*10
Result
m: scalaSci.Mat = 
 -1.000  -1.000  -1.000  -1.000  
-1.000  -1.000  -1.000  -1.000  
-1.000  -1.000  -1.000  -1.000  
mm: scalaSci.Mat = 
-5.000  -5.000  -5.000  -5.000  
-5.000  -5.000  -5.000  -5.000  
-5.000  -5.000  -5.000  -5.000  
Note that m is also decremented.
